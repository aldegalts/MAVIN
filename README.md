Веб-сервис предзаказа еды с самовывозом (Click & Collect) и персональным ИИ-ассистентом по питанию.

## О проекте

MAVIN — платформа, которая позволяет пользователям заранее выбирать блюда в ресторанах и забирать готовый заказ без ожидания. Доставка курьером не предусмотрена — только самовывоз.

Ключевые возможности:

- **Каталог ресторанов и меню** с фильтрацией по категориям, рейтингу и аллергенам
- **Корзина и оформление заказа** с выбором времени самовывоза и способа оплаты
- **ИИ-ассистент** на базе OpenRouter — подбирает блюда под вкусы и ограничения пользователя
- **Подписка** для получения скидок на заказы
- **Промокоды** с одноразовым применением
- **Панель администратора ресторана** — управление меню, заказами, акциями
- **Панель администратора сайта** — управление пользователями, ресторанами, подписками

## API

Базовый URL: `/api/v1`  
Авторизация: `Authorization: Bearer <token>` (JWT)  
Формат: JSON

Основные группы эндпоинтов:

| Группа | Описание |
|---|---|
| `POST /auth/register`, `POST /auth/login` | Регистрация и вход |
| `GET /auth/me`, `PUT /profile` | Профиль пользователя |
| `GET /restaurants` | Каталог ресторанов |
| `GET /restaurants/{id}/menu` | Меню ресторана |
| `POST /cart`, `GET /cart` | Корзина |
| `POST /orders`, `GET /orders` | Заказы |
| `POST /ai/recommend` | ИИ-ассистент |
| `GET /subscriptions`, `POST /subscriptions/subscribe` | Подписка |
| `POST /promo-codes/apply` | Промокоды |
| `POST /upload` | Загрузка изображений |

Полный список: [`docs/list-of-API-methods.md`](./docs/list-of-API-methods.md)  
OpenAPI-схема: [`docs/swagger.json`](./docs/swagger.json)

## Методология разработки

Проект ведётся по адаптированному **Scrum**: спринты по 1 неделе, ежедневные встречи (созвоны) по 15 минут.

**Git Workflow:**
- Основная ветка разработки: `develop`
- Ветки: `feature/<описание>`, `fix/<описание>`, `hotfix/<описание>`
- Коммиты: [Conventional Commits](https://www.conventionalcommits.org/) на английском (`feat:`, `fix:`, `refactor:`, `docs:`)
